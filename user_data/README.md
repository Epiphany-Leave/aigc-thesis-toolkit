# User Data

Put personal reference materials here.

This directory is intentionally ignored by git, except for this README. Typical contents:

- reference images
- papers and bibliography files
- source code used as reference
- simulation files
- drawings
- physical prototype photos
- school-specific Word templates or report examples

The reusable workflow should not depend on any specific file in this directory. Update `configs/default.yaml` or `thesis/style.md` if your project needs to refer to these materials.

`resources.md` is generated by the workflow. Run:

```bash
python workflow.py resources --overwrite
```

or run `python workflow.py outline`, which refreshes the resource index before generating the outline.
